## Important Topics

In Nominative, Possessive Pronomen are always based on the Subject but ending (if it ends on e or not) is based on the Object.

For Example:

- Das ist mein Auto (This is my Auto).
- Das ist meine Flasche (This is my Flasche).
- Das ist sein Auto (This is his Bottle).
- Das ist seine Flasche (This is his Bottle).

|          | Nominative Possessive Pronomen |
| -------- | ------------------------------ |
| Ich      | mein                           |
| du       | dein                           |
| er/es    | sein                           |
| sie      | ihr                            |
| wir      | unser                          |
| ihr      | euer                           |
| Sie (pl) | ihr                            |
| Sie      | Ihr                            |
